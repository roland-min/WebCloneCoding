Git

2024-12-21 10:00:00

Git이란?

  • Git은 분산 버전 관리 시스템(DVCS)입니다.
  • 주요 기능: 코드 히스토리 관리, 협업 지원, 변경 내용 추적.
  • Linus Torvalds에 의해 개발되었습니다.
  • 무료로 제공되며 오픈 소스 소프트웨어입니다.
  • 로컬 저장소와 원격 저장소를 활용하여 안정적인 버전 관리를 지원합니다.

Git의 주요 명령어

  • git init: 새로운 Git 저장소를 초기화합니다.
  • git clone: 원격 저장소를 로컬로 복제합니다.
  • git add: 변경된 파일을 스테이지에 추가합니다.
  • git commit: 변경 내용을 로컬 저장소에 저장합니다.
  • git push: 로컬 변경 내용을 원격 저장소에 반영합니다.
  • git pull: 원격 저장소의 변경 내용을 가져옵니다.
  • git status: 현재 저장소 상태를 확인합니다.
